  * pyTorch
* Python可视化  
  * [Matplotlib](https://nbviewer.jupyter.org/github/haochen95/python_tutorial/blob/master/Python_Visualize/Scikit-learn-matplotlib-bar.ipynb)

* Python正则表达式
